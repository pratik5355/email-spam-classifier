# Email Spam Classifier

Hey there! This is a simple machine learning project I put together to classify emails into spam or ham (not spam). 

## Dataset

I used the Spam Assassin dataset to train the model. If you want to check it out or grab the data yourself, you can find it on Kaggle here:
[Spam Assassin Email Classification Dataset](https://www.kaggle.com/datasets/ganiyuolalekan/spam-assassin-email-classification-dataset)

## What's inside

- `email_spam_classifier.ipynb`: The Jupyter Notebook where the magic happens—data exploration, preprocessing, and model training.
- `spam_assassin.csv`: The dataset used for this project.

Feel free to poke around the code and experiment with it!
